Зачем нужен break?
Экономия времени: Позволяет прекратить выполнение цикла, как только достигается нужное условие, избегая ненужной обработки оставшихся элементов.
Контроль потока выполнения: Иногда требуется немедленно завершить цикл, не дожидаясь завершения его полного прохода.
Ускорение алгоритмов: В некоторых случаях, особенно при работе с большими наборами данных, использование break значительно ускоряет работу программы.
example:

package main

import "fmt"

func main() {
    numbers := []int{1, 3, 5, 6, 9, 10}

    for _, number := range numbers {
        if number%2 == 0 {
            fmt.Println("Первое четное число:", number)
            break
        }
    }
}
